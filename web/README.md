# Getting Started

- You need NodeJS and Yarn.
- Create a file named `.env` with the API keys for OpenAI, Azure Speech API and Assembly AI keys

```
OPENAI_API_KEY=...
AZURE_API_KEY=...
AZURE_REGION=...
ASSSEMBLYAI_API_KEY=...
```

- Start the project with `yarn dev`
